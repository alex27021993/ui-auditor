# UI Auditor

Набор скилов для **глубокого QA/UX-аудита интернет-магазинов** через Claude Code.
Два трека:

- **`ecommerce-functional-audit`** (Трек A) — работает ли и логично ли: навигация, поиск,
  каталог, ГЛУБОКИЙ аудит фильтров/сортировки, варианты товара, корзина, чекаут, оплата,
  вишлист, аккаунт, формы, промо/сток, 404. Состояние читаем из DOM, без скриншотов.
- **`ecommerce-visual-audit`** (Трек B) — как сайт выглядит: вёрстка, адаптив (десктоп + мобайл),
  типографика, контент-консистентность. Скриншоты разрешены.

Методика и принципы — в [CLAUDE.md](CLAUDE.md). Детали проверок — в `skills/<name>/SKILL.md`.

## Установка (для коллег)

Склонировать в папку скилов Claude Code:

```bash
git clone https://github.com/<owner>/ui-auditor.git ~/.claude/skills/ui-auditor
```

Claude Code сам подхватит скилы из этой папки.

## Обновление до последней версии

```bash
cd ~/.claude/skills/ui-auditor && git pull
```

Рекомендуется делать `git pull` **перед каждым прогоном** — так всегда свежая методика.
Чтобы получать уведомления о новых версиях — нажмите **Watch → All Activity** на репозитории.

## Использование

В чате Claude Code:

```
Проведи функциональный аудит интернет-магазина: https://example.com
```

или явно сослаться на скил `ecommerce-functional-audit` / `ecommerce-visual-audit`.

## Правки

Канонический источник — этот репозиторий. Изменения принимаются только через Pull Request
(ветка `main` защищена). Отчёты прогонов (`audit-*.xlsx` / `audit-*.md`) в репозиторий
не коммитятся — см. `.gitignore`.
